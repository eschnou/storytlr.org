Get started

You have two options:

  • 1. Install on your own server

    You can download and install storytlr on most web servers. Check the installation instructions to setup Storytlr and solve common issues. The following commercial web hosts offer single click Storytlr installation*: Fused Network

    2. Register on a live service

    You can register on a full fledged Storytlr service. No installation required! The following sites offer registration*: Betavine Chronicle

    * We have not tried out these hosts/services ourselves, we do not endorse nor get any money from them, but list them for your convenience. Please tell us about your experiences or in case you want to be listed here.
