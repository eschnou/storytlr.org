Get started

You have two options:

  • 1. Install on your own server

    You can download and install storytlr on most web servers. Check the installation instructions to setup Storytlr and solve common issues.

    2. Register on a live service

    You can register on a full fledged storytlr service. No installation required!

Join the community

    Storytlr is open source and everyone can help to make it better.

  • Report ideas and issues
    Have a cool new idea or found a bug? Please let us know.

  • Join the conversation
    We are discussing various topics on the Storytlr mailinglist. We value your opinion.

  • Play with code
    Our project page on Google Code has our latest stable releases, a wiki and issue tracker. You can checkout work in progress from GitHub.
